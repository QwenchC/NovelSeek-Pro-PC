🚀 快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. 安装依赖 (已完成)
     npm install ✓

  2. 启动开发服务器
     npm run tauri:dev

     ⚠️  首次运行需要 10-20 分钟编译 Rust 依赖

  3. 配置 API
     - 打开应用后进入"设置"页面
     - 输入 DeepSeek API Key
     - 测试连接

  4. 开始创作
     - 创建新项目
     - 添加章节
     - 使用 AI 生成内容
